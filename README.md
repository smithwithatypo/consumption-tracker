# Consumption Tracker
Track your daily consumption to figure out what affects you good or bad

## Stack
- Frontend: Vite + React + shadcn
- Backend: Go + Chi
- Database: PostgreSQL
- LLM: Anthropic API
- Deployment: Railway

## Development

Run both servers:
``` bash
make dev
``` 

Run individually:
``` bash
make frontend 
make backend 
```
